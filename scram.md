# Interpretation

- Scramjets are **engine‑efficient** (4–8× Isp, 6–7× range per kg fuel).  
- Scramjet systems are **only mildly more efficient** (1.2–1.5×) because the booster dominates mass.  
- Rocket planes are **far more flexible** (0–space, no booster).  
- Scramjets win only in **sustained hypersonic atmospheric cruise**.

## 5. Worked Example: Isp Advantage in Ramjet Regime

Take a **Mach‑3 cruise** segment:

- Ramjet Isp ≈ 1100 s  
- Rocket Isp ≈ 300 s  



\[
\frac{1100}{300} \approx 3.7
\]



**Ramjet delivers ~3.5–4× more impulse per kg of fuel in Mach‑3 atmospheric cruise than a rocket plane.**

Now include the accelerator/booster:

- Booster/accelerator Isp ≈ 250 s  
- Ramjet phase Isp ≈ 1100 s  
- Assume half the propellant is in the booster, half in the ramjet phase (simplified).  

Effective system Isp (very rough):



\[
I_{sp,\text{eff}} \approx \frac{0.5 \cdot 250 + 0.5 \cdot 1100}{1} = 675 \text{ s}
\]



Compare to rocket plane:



\[
\frac{675}{300} \approx 2.25
\]



So the **engine‑level advantage (~3.7×)** collapses to a **system‑level advantage of ~2×** once you pay for the accelerator.

### Effective system Isp (very rough weighted average)


## Effective Isp Calculation (GitHub‑Safe Formatting)

We define:

- fb = 0.35   (booster propellant fraction)
- fr = 0.35   (ramjet propellant fraction)
- fs = 0.30   (scramjet propellant fraction)

- Isp_b = 250 s    (booster)
- Isp_r = 1100 s   (ramjet)
- Isp_s = 1500 s   (scramjet)

Effective Isp is the weighted sum:

    Isp_eff = fb * Isp_b  +  fr * Isp_r  +  fs * Isp_s

Substitute values:

    Isp_eff = (0.35 * 250)  +  (0.35 * 1100)  +  (0.30 * 1500)

Compute each term:

    0.35 * 250  =   87.5
    0.35 * 1100 =  385
    0.30 * 1500 =  450

Add them:

    Isp_eff = 87.5 + 385 + 450
    Isp_eff = 922.5 seconds

Comparison to a rocket plane (Isp ≈ 300 s):

    Ratio = Isp_eff / Isp_rocket
    Ratio = 922.5 / 300
    Ratio ≈ 3.1

So the stacked booster + ramjet + scramjet system
has roughly **3.1× the effective Isp** of a rocket plane
for an atmospheric hypersonic mission.

So the **stacked ramjet+scramjet system can be ~3× better in effective Isp** than a pure rocket plane for a long atmospheric hypersonic mission.

## 4. Mission‑Level Interpretation

- **Where the stacked airbreather wins hard:**
  - **Long‑range hypersonic cruise** (Mach 3–8) with **no need for vacuum operation**.  
  - You get **roughly 2–3.5× better effective Isp** and **4–8× more km per kg of fuel** than a rocket plane.  
  - Payload fraction for atmospheric missions can be **2–4× better**.

- **Where the rocket plane still wins:**
  - **Simplicity:** one engine cycle, no hand‑off between booster/ramjet/scramjet.  
  - **Full envelope:** from **zero speed to vacuum** with no dependence on air density.  
  - **True space access:** stacked airbreathers still need a big rocket stage to go orbital; the advantage collapses.

# Bottom Line

Piggybacking **scramjet on ramjet on booster** turns into a **very fuel‑efficient hypersonic cruiser**—on the order of **2–3.5× better effective Isp** and **multiple‑times better range per kg fuel** than a rocket plane, **as long as you stay in the atmosphere and spend real time in the Mach‑2–10 band**.

The moment the mission demands **true space access or simple, robust operations**, the **rocket plane’s brute‑force flexibility** wins back the advantage.
